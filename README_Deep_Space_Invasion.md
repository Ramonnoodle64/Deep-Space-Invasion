# 🚀 Deep Space Invasion

A 2D arcade-style space shooter built with **Python** and **Pygame**
featuring multiple enemy types, boss battles, pixel-perfect collision
detection, original pixel art, and sound effects.

## Features

-   Player-controlled spaceship
-   Multiple enemy ship types
-   Two unique boss encounters
-   Pixel-perfect collision detection using Pygame masks
-   Health bars for player and bosses
-   Sound effects
-   Original pixel-art sprites
-   Object-oriented game architecture
-   Interactive menu buttons and UI elements

## Technologies

-   Python 3
-   Pygame

## Programming Concepts Demonstrated

-   Object-Oriented Programming (OOP)
-   Inheritance
-   Encapsulation
-   Event-driven programming
-   Collision detection
-   Game loop architecture
-   Modular programming

## Installation

``` bash
pip install pygame
python main.py
```
